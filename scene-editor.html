Device Inputs

If a device contains an invalid field value, the game will kick you when you move or copy that device, or when you open the device and click Apply. This editor tries to prevent invalid values from being entered, but if you experience unexpected kicks, check the affected device's fields for out-of-range or missing values.

Rotation bug

There is a visual issue between this editor and PUBG where some rotations/transforms will look slightly desynchronized between the two, and may cause some objects to appear rotated in game differently than in the editor. I think this is due to some internal precision or rounding issue in game or maybe the wrong origin for rotations im not sure.
